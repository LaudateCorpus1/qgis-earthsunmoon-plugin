Earth, Sun, Moon, and Planets Plugin

This plugin uses the highly accurate Skyfield library to show where the sun, moon, and planets are located at their azimuth from earth's perspective for a particular date and time. An additional algorithm calculates lunar and solar data for a particular location on earth given a date and time. When installed, this plugin can be found in the QGIS menu under Plugins->Earth, sun, moon & plents. The planetary location alogorims are as follows:

  • Sun position directly overhead - This shows the location of the sun where it is directly overhead for a paricular date and time.
  • Moon position directly overhead - This shows the location of the moon where it is directly overhead for a paricular date and time.
  • Planetary positions directly overhead - This shows the location of the planets where they are directly overhead for a paricular date and time.
